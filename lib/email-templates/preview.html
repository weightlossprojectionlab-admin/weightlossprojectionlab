Email Notification Templates Preview

1. Medication Added Email
Notifies family members when a new medication is added to a patient's profile. Features blue color coding, full medication details, and direct link to medications tab.
2. Weight Vital Logged Email
Notifies family members when a new weight reading is logged. Features large emoji display, prominent value, and optional notes field.
3. Lab Results Document Uploaded Email
Notifies family members when a new lab results document is uploaded. Features lab flask icon, blue color coding, and file size information.
4. Monthly Health Report Email
Notifies family members when a health report is generated. Features chart icon, summary section, and key findings list.
