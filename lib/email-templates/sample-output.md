## Design Consistency

All templates share:

1. **Header Design**
   - Purple gradient (135deg, #667eea to #764ba2)
   - White text with 28px title
   - "Weight Loss Projection Lab" subtitle
   - Rounded top corners (10px)

2. **Content Container**
   - White background
   - 30px padding
   - Light gray border (#e5e7eb)
   - Rounded bottom corners (10px)

3. **Typography**
   - System font stack (-apple-system, BlinkMacSystemFont, Segoe UI, etc.)
   - 1.6 line height for readability
   - Clear hierarchy (h1: 28px, h2: varies, body: 16px)

4. **Color Palette**
   - Primary: #667eea (purple)
   - Secondary: #764ba2 (darker purple)
   - Info: #3b82f6 (blue)
   - Warning: #f59e0b (orange)
   - Danger: #ef4444 (red)
   - Success: #10b981 (green)

5. **Interactive Elements**
   - Purple gradient buttons
   - White text on buttons
   - 15px vertical, 40px horizontal padding
   - 8px border radius
   - 600 font weight

6. **Footer**
   - Gray text (#9ca3af)
   - 12px font size
   - Copyright notice
   - Auto-notification disclaimer

7. **Responsive Design**
   - 600px max width
   - Works on mobile and desktop
   - Table-based layout for document icons

8. **Context-Aware Links**
   - Direct deep links to patient tabs
   - Query parameters for tab selection
   - Full report URLs for health reports

## Template Summary

Created **4 main email templates** with **8 example variations**:

| Template | Purpose | Key Features |
|----------|---------|--------------|
| **Medication Change** | Notify about med changes | Color-coded by action (add/update/delete), full medication details, prescriber info |
| **Vital Logged** | Notify about new vitals | Large emoji display, type-specific colors, optional notes field |
| **Document Uploaded** | Notify about new docs | Category icons, file size, document metadata |
| **Health Report** | Notify about reports | Summary section, key findings list, report date |

Each template returns:
- `html`: Fully formatted HTML email
- `text`: Plain text fallback version
- `subject`: Context-aware subject line

All templates are production-ready and match the existing family invitation email styling.
