# WPL Brand Logos

This directory contains official brand logos for Weight Loss Projection Lab.

## Available Formats

### Full Color Logos
- `wlpl-logo-full-color.svg` - Primary logo, full color (SVG vector)
- `wlpl-logo-full-color.png` - Primary logo, full color (PNG, 2000x2000px)
- `wlpl-logo-full-color-wide.png` - Horizontal logo (PNG, 2000x600px)

### White Logos (for dark backgrounds)
- `wlpl-logo-white.svg` - White logo (SVG vector)
- `wlpl-logo-white.png` - White logo (PNG, 2000x2000px)

### Black Logos (for light backgrounds)
- `wlpl-logo-black.svg` - Black logo (SVG vector)
- `wlpl-logo-black.png` - Black logo (PNG, 2000x2000px)

### Monochrome Logos
- `wlpl-logo-grayscale.svg` - Grayscale logo (SVG vector)
- `wlpl-logo-grayscale.png` - Grayscale logo (PNG, 2000x2000px)

## Usage Guidelines

1. **Minimum Size**: Do not display the logo smaller than 120px wide
2. **Clear Space**: Maintain minimum clear space of 1/4 logo height on all sides
3. **Color Backgrounds**: Use white logo on dark backgrounds, colored logo on light backgrounds
4. **Do Not**:
   - Modify logo colors
   - Distort or stretch the logo
   - Add effects (shadows, gradients, etc.)
   - Use low-resolution versions in print
